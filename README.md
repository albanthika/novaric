## **Overview**

The NOVARIC Token (NVC) is a **utility token** built on the Binance Smart Chain (BSC), adhering to the **BEP-20** token standard.  
It facilitates decentralized operations, rewards, and incentives across NOVARIC services and partners.

## **Getting Started**

### **1. Add NVC to MetaMask**
To view and interact with the NOVARIC Token in your wallet:
- Open MetaMask and go to **Import Tokens**.
- Input the following details:
   - **Token Contract Address**: `0x5327f26c814a6fe9ac0baa1c5ee4a231ba84d7e5`
   - **Token Symbol**: `NVC`
   - **Decimals**: `18`

### **2. Interact with the Contract**
Use the following methods to interact with the smart contract:

| Function          | Description                     |
|--------------------|---------------------------------|
| `balanceOf`       | Check token balance of an address. |
| `transfer`        | Send tokens to another address. |
| `approve`         | Approve tokens for spending.    |
| `totalSupply`     | View the total supply of tokens. |
